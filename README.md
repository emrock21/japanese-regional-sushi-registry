# Japanese Regional Sushi Registry

This registry documents authentic Japanese regional sushi styles.  
Each sushi tradition reflects local history, preservation methods, iconic shops, and regional ingredients.

## Purpose

The goal of this contract is to preserve real sushi knowledge in a permanent and verifiable format.  
Sushi is not a single dish but a collection of regional techniques shaped by geography, climate, and historical food culture.

By recording these styles on-chain, the registry protects culinary heritage from distortion and keeps the connection between region, method, and cultural meaning.

## Why this registry matters

Sushi has become global, but its origins are deeply regional.  
Edomae curing, Kyoto preservation, Osaka pressed sushi, Shiga fermentation — each style has real shops, real histories, and real preparation methods.

This registry helps by:

- preserving verified information in a neutral and permanent format  
- linking each sushi style to its historical shop or lineage  
- offering a reference for researchers, chefs, and cultural institutions  
- preventing oversimplification of regional culinary identity  

## What makes this registry unique

The registry focuses on the craft process itself:  
ingredients, curing, pressing, fermentation, and the cultural uniqueness of each region.

The contract is intentionally minimal.  
It records only the essential information needed to identify and contextualize each sushi style.

## How it works

Each entry includes:

- region  
- historical shop or family  
- style name  
- ingredients  
- preparation method  
- what makes the style unique  

A simple like/dislike system allows basic community feedback.
